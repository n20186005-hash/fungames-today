# 荒野沙虫大逃亡 (Desert Run: Spice Wars)

# 开发者是谁？
这款游戏由独立网页游戏开发者制作，专注于构建轻量级、无需下载的 HTML5 浏览器游戏，旨在为玩家提供即点即玩的快节奏动作体验。

# 描述
《荒野沙虫大逃亡》是一款紧张刺激的横版无尽跑酷生存游戏。在这个荒凉的沙漠世界中，巨大的地下沙虫（受到经典科幻作品《沙丘》启发）时刻威胁着地面的安全。玩家需要驾驶一辆经过改装的沙漠越野卡车，在危机四伏的沙丘上飞驰。你的任务不仅仅是生存，还要尽可能多地收集珍贵的“香料”（Spice），拯救受困的幸存者，并利用收集到的资源在商店中购买各种生存装备。

# 如何玩？
游戏的目标非常简单：活下去，跑得越远越好。
1. **驾驶与躲避**：控制卡车在沙漠中左右移动，避开突然钻出的巨大沙虫。
2. **跳跃机制**：遇到障碍物或沙虫攻击时，及时跳跃躲避。
3. **资源收集**：
   - **香料 (金色晶体)**：这是游戏中的货币，用于购买道具。
   - **幸存者 (蓝色小人)**：拯救他们可以增加你的分数和卡车上的乘员数量。
   - **道具**：收集红心（恢复生命）和护盾（提供无敌时间）。
4. **商店系统**：随时按“P”键暂停游戏进入商店，使用香料购买急救包或能量护盾。

# 在线操控指南
游戏设计了直观的控制方式，支持电脑和移动设备，无需复杂的操作即可享受乐趣。界面上方会实时显示生命值、香料数量、获救人数以及行驶距离。

# 一些有趣之处
- **沙虫威胁**：沙虫有两种攻击模式——一种是在地下潜伏突然冲出，另一种是高高跃起从天而降，玩家需要根据沙虫的警告信号（感叹号）快速反应。
- **动态车队**：随着你拯救的幸存者越来越多，你会看到他们出现在你的卡车后斗上，看着车队壮大非常有成就感。
- **无敌模式**：对于只想放松看风景或练习的玩家，游戏开始前提供了一个“无敌模式（Infinite Mode）”选项，勾选后你将不会死亡。
- **多语言支持**：游戏内置了英文、繁体中文和简体中文，点击主界面的语言按钮即可一键切换。

# 主要特点
- **复古像素风格**：简洁明快的图形风格，配合流畅的视差滚动背景（山脉、太阳）。
- **商店升级系统**：通过收集资源来购买“维修工具包”和“能量护盾”，增加了游戏的策略性。
- **高分记录**：游戏会自动保存你收集的香料总量和最远行驶距离记录。
- **合成音效**：独特的复古合成器音效，为跳跃、收集和碰撞提供听觉反馈。

# 卓越成功
在游戏中，真正的成功不仅仅是跑得远。卓越的玩家能够：
- 积累大量的**香料**财富。
- 保持卡车满血状态，并时刻开启**能量护盾**。
- 拯救超过 **5 名以上的幸存者**（把卡车后座填满）。
- 在不使用“无敌模式”的情况下突破 **10000米** 的行驶距离。

# 关于常见问题解答
- **Q: 游戏需要下载吗？**
  A: 不需要，这是一款纯网页游戏，在浏览器中打开即可游玩。
- **Q: 如何购买道具？**
  A: 在游戏中按下“P”键或点击暂停按钮，即可打开商店界面使用香料购买。
- **Q: 撞到沙虫会怎样？**
  A: 如果没有护盾，你会损失一颗心。生命值归零则任务失败。
- **Q: 游戏进度会保存吗？**
  A: 是的，游戏会自动在本地保存你的香料总量和最高分记录。

# 玩法
核心玩法是“跑酷+资源管理”。
随着行驶距离的增加，游戏速度会逐渐加快，沙虫的生成频率也会变高。你需要学会在高速移动中预判危险，同时贪心地去吃掉路上的每一个香料和道具。如果你受损了，记得权衡是继续冒险收集香料，还是赶紧暂停购买维修包。
