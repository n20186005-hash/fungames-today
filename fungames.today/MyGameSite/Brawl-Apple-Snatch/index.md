# 游戏名称
好友大乱斗：吃苹果 (Brawl: Apple Snatch)

# 开发者是谁？
由独立游戏开发者精心制作，旨在为您带来纯粹的街机对战乐趣。

# 描述
《好友大乱斗：吃苹果》是一款快节奏、复古像素风格的休闲竞技游戏。在这个色彩斑斓的世界里，您的目标非常简单：比对手吃掉更多的苹果！游戏不仅考验您的反应速度，还需要您灵活运用策略，利用地形和体型优势将对手撞飞。无论是独自挑战高智商的电脑对手，还是与好友同屏一较高下，这都是一款让您手心出汗、爆笑不断的派对游戏。

# 如何玩？
1. **选择模式**：您可以选择单人挑战（对抗 AI 电脑）或双人对战（本地同屏）。
2. **抢夺苹果**：红色的美味苹果从天而降，吃到它们可以增加分数并让您的角色**变大**！
3. **躲避毒药**：千万小心紫色的毒苹果！吃到它不仅会扣分，还会让您的角色**变小**并变得脆弱。
4. **互相攻击**：利用体型优势撞击或攻击对手，让他们眩晕或掉出地图，从而获得巨大优势。
5. **赢得胜利**：在倒计时结束前，分数最高的玩家获胜！

# 在线操控指南
游戏设计了简单直观的操作方式，让您能立刻上手：

### 电脑端 (键盘操作)

**玩家 1 (蓝色角色 / 左侧):**
* **W 键**: 跳跃 (在空中按两次可进行二段跳)
* **A 键**: 向左移动
* **D 键**: 向右移动
* **S 键**: 攻击 / 冲撞

**玩家 2 (红色角色 / 右侧):**
* **↑ (上箭头)**: 跳跃 (支持二段跳)
* **← (左箭头)**: 向左移动
* **→ (右箭头)**: 向右移动
* **↓ (下箭头)**: 攻击 / 冲撞

**通用功能:**
* **P 键**: 暂停游戏
* **空格键**: 也可以用于玩家1跳跃

### 移动端 (手机/平板)
* **屏幕左侧**: 虚拟方向键控制左右移动。
* **屏幕右侧**: 设有大号的“跳跃”和“攻击”按钮。
* **触控优化**: 完美支持多点触控，操作丝滑流畅。

# 玩法
游戏的核心在于“争夺”与“生存”。不仅仅是简单的收集游戏，您必须时刻关注对手的位置。通过连续吃掉红苹果，您的角色体型会显著增大，此时您的推挤力量也会变强，可以轻松将瘦小的对手撞下悬崖。但要小心，体型变大也意味着更容易触碰到毒苹果或遭受攻击。每一局都充满了变数！

# 一些有趣之处 
* **体型变化机制**：这不仅仅是视觉效果！吃得越多，您就越重、越强壮；中毒越多，您就越轻、越容易被击飞。
* **动态地图**：游戏包含多个关卡，从经典的平地金字塔，到危险的浮空岛，再到封闭的角斗场，每一关都需要不同的策略。
* **物理碰撞**：看着对手被您一记重拳打飞出屏幕，或者因为太贪吃而掉进深渊，总是充满了滑稽的乐趣。
